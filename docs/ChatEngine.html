              connect(uuid, state, authKeyopt, authDataopt)

              Source:
              • chat-engine/src/bootstrap.js, line 96

              Connect to realtime service and create instance of Me

              Parameters:
              Name Type Attributes Description
              uuid String

              A unique string for Me. It can be a device id, username, user id, email, etc.
              state Object

              An object containing information about this client (Me). This JSON object is sent to all other clients on the network, so no passwords!
              authKey String <optional>

              A authentication secret. Will be sent to authentication backend for validation. This is usually an access token or password. This is different from UUID as a user can have a single UUID but multiple auth keys.
              authData Object <optional>

              Additional data to send to the authentication endpoint to help verify a valid session. ChatEngine SDK does not make use of this, but you might!
